# SocketMe

Simple android app to send sensor information to a remote websocket server.

Current Services:
 - battery
 - location
 - motion
 - orientation
 - signal (gsm)
 - wifi (scanner)
 - doppler (Not working yet)
 
Device Deployment (with livereload):
```
npm run dev
```

Local serve (with livereload):
```
npm run serve 
```
